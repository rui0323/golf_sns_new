# <ゴルフ×SNS>

## サイト概要
### サイトテーマ
ゴルフを通じて多くの人たちと関わることができるSNS

### テーマを選んだ理由
趣味でゴルフをしていますが同世代の人たちや知り合いでゴルフをしている人が少ないため広範囲で共有や仲間を作れるコミュニティの場があるといいなと思ったため

### ターゲットユーザ
ゴルフをする人・これから始めようかと思っている人

### 主な利用シーン
自分が行ったゴルフ場の情報を投稿するとき、他のプレーヤーがどんな場所を投稿しているかを見たいとき
